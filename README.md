## Credit Card Approval System ##

## Project Overview
The Credit Card Approval System is a Machine Learning-based web application that predicts whether a credit card application is likely to be approved based on applicant details. The system uses a trained classification model and provides instant prediction through a simple web interface.

## Features
- Predicts credit card approval status
- User-friendly web interface
- Machine Learning based prediction
- Fast and accurate results
- Built using Flask

## Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn
- HTML
- CSS
- JavaScript
- Joblib
